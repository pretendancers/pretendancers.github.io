*包装清单*

拆开塑封抽出封套，翻开磁吸盖后应有如下内容：

- 键盘外壳（上下盖拧好螺丝塑封）

- PCB母板（静电袋包装）

- 定位板（长自封袋包装）

- 小自封袋A，内含

	- M2螺丝 x14

	- M3螺丝 x8

- 小自封袋B，内含

	- 硅胶粒 x12

	- 遮光棉（环形，背胶） x3

	- 短排线（16.5mm，8pin反向，0.5mm间距） x3

- 小自封袋C，内含

	- PCB c口子板 x1

- 大自封袋A，内含

	- 锂电池 x2

- 大自封袋B，内含

	- PCB 磁吸子板 x1

	- 长排线（60mm，8pin反向，0.5mm间距）x3

	- 磁吸盖板（黑色fr4） x2

	- 磁吸小板底棉（条状，背胶） x3

- 1.25mm 磁吸内六角扳手 x1

- 2.5mm 磁吸内六角扳手 x1

- PCB孔位参考卡（背面有本站二维码） x1
